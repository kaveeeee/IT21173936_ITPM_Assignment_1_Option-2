# Pixelssuite – Functional and Usability Testing Suite

## Project Overview

This project contains a **functional and usability test suite** for the website [https://www.pixelssuite.com/](https://www.pixelssuite.com/),

The assignment assesses the functional correctness and usability of ten key features of the Pixelssuite website. It includes **36 manually designed test scenarios** and **one automated test scenario** built using **Python** and **Playwright** to verify the image preview functionality of a selected feature using a PNG image file.

## Features Tested

The following ten features of [pixelssuite.com](https://www.pixelssuite.com/) were tested:

5. Compression
6. Image Format Conversion
7. Meme Generation
